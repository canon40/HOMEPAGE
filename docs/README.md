# 홈페이지 첨부 자료 (Attachments)

이 폴더는 바이어 자료실 다운로드 및 코팅제 추천(Finder) 관련 자료를 두는 곳입니다.

---

## 바이어 전용 자료 (Buyer Documents)

아래 PDF 파일을 이 폴더에 넣으면 홈페이지 **바이어 전용 자료실** 섹션의 다운로드 링크가 연결됩니다.

- `permacoat_manual_ko.pdf` — 사용설명서 (PDF)
- `permacoat_moq_price_ko.pdf` — MOQ 단가표

파일을 추가한 뒤 저장하면 링크가 자동으로 동작합니다.

---

## 코팅제 추천 (Finder) 관련 자료

**코팅제 추천**은 홈페이지 내 5문항 질문 기반으로 퍼마코트 등급(QUICK / TITAN / RESIN)을 추천하는 기능입니다.  
추가로 배포할 Finder 안내 PDF나 설명 자료가 있으면 이 `docs/` 폴더에 넣고, 필요 시 `index.html` 바이어 자료실 근처에 링크를 추가하면 됩니다.

- 예: `finder_guide_ko.pdf` — Finder 사용 안내 (필요 시 추가)
